# display-names
Converts names to a human readable format. <br /><br />

Examples:<br />
"JohN SmiTh"       ->   "John Smith" <br />
"bob barker"       ->   "Bob Barker" <br />
"tod O'henry"      ->   "Tod O'Henry" <br />
"bILLY jAmEs"      ->   "Billy James" <br />
"ALBERT A JAckson" ->   "Albert A Jackson" <br />
"cher"             ->   "Cher" <br />
